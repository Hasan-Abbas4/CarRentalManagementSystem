#  Car Rental Management System

##  Overview

A full-featured Car Rental Management System built using ASP.NET MVC.
This system provides a complete solution for managing car rentals, including booking, user management, admin controls, and additional features like coupons, drivers, and damage reporting.

---

##  Features

###  Customer Features

* User Registration & Login
* Browse available cars
* Book cars with date selection
* View booking history
* Add cars to favorites 
* View favorite cars list
* Report car damage
* Apply coupons (if available)
* Dashboard with booking insights

---

###  Admin Features

* Admin Dashboard with system overview
* Manage Cars (Add, Update, Delete)
* Manage Users
* Manage Bookings (Approve/Reject)
* Manage Drivers (Add, View)
* Manage Coupons (Create, Activate/Deactivate)
* View All Bookings
* View Reported Damages
* Manage Damage Reports
* Assign Drivers (if implemented)

---

##  Tech Stack

* ASP.NET MVC (.NET Framework)
* SQL Server
* Entity Framework
* Bootstrap
* jQuery

---

##  Project Structure

* Controllers → Handle business logic
* Models → Database structure
* Views → UI (Razor)
* App_Start → Configurations
* Content → CSS & UI
* Scripts → JavaScript files

---

##  How to Run the Project

1. Clone the repository
2. Open in Visual Studio
3. Configure database connection in `web.config`
4. Apply migrations / setup database
5. Run the project
